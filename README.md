# 🧠 Quiz Game Website

An interactive and responsive **Quiz Game Website** built using **HTML, CSS, and JavaScript**. The application fetches real-time quiz questions from the Open Trivia Database API and provides a fun learning experience with multiple categories, a countdown timer, score tracking, and high-score storage.

---

## ✨ Features

### 🎯 Multiple Quiz Categories

Choose from different categories:

* General Knowledge
* Science
* Sports
* History

### 🌐 Live Questions from API

Questions are fetched dynamically using the Open Trivia Database (OpenTDB) API.

### ⏳ Countdown Timer

* 15 seconds per question
* Automatically disables options when time runs out

### 📊 Score Tracking

* Displays final score after quiz completion
* Calculates score in real-time

### 🏆 High Score System

* Saves highest score using Local Storage
* High score persists even after page refresh

### ✅ Answer Review

At the end of the quiz, users can view:

* Their selected answers
* Correct answers
* Correct/Wrong answer indication

### 🎨 Modern Dark Theme UI

* Dark mode design
* Smooth animations
* Responsive layout
* Attractive buttons and typography

## 🎮 How to Play

1. Select a quiz category.
2. Click **Start Quiz**.
3. Read the question carefully.
4. Select the correct answer before the timer reaches zero.
5. Click **Next Question** to continue.
6. After completing all questions, view your final score and answer review.

---

## ⏱️ Quiz Rules

* Each quiz contains **5 questions**.
* Each question has **4 options**.
* Only one answer is correct.
* Each question must be answered within **15 seconds**.
* Unanswered questions are treated as incorrect.
* Final score is displayed after all questions are completed.
